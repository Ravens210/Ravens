Keterangan Pengalaman

  • 1. juara 3 Olimpiade IPA SD tingkat kabupaten.

    juara 3 Olimpiade IPA SD tingkat kabupaten itu saya ketika saya kelas 5 untuk pertama kalinya saya mengikuti lomba. hal yang membuat saya terkejut bisa langsung dapat juara 3 padahal persiapannya kurang masksimal.

  • 2. Peserta Olimpiade IPA SD tingkat kabupaten

    ketika saya kelas 6 saya dipanggil lagi buat ikut lomba olimpiade cabang IPA. Tidak seperti tahun sebelumnya, saya membuat persiapan lebih matang. Namun, takdir berkata lain, saya tidak berhasil mendapat juara. Saya tidak terlalu sedih karena sudah pertama juara sebelumnya.

  • 3. Peserta cerdas cermat antar sekolah di MUQ PIDIE

    Pada tahun tersebut MUQ Pidie mengadakan ulang tahunnya yang diadakan oleh pemerintah dengan lomba berbagai cabang salah satunya cerdas cermat. saya dan teman-teman mengikuti lomba tersebut, selama lomba tersebut berlangsung tim kami nyaris masuk final. Namun, karena tidak kemampuan kami menjawab soal terakhir, kami berakhir kalah.

  • 4. Peserta Tahfiz Quran MTQ kabupaten

    Saat itu, MTQ tingkat kabupaten ketika masa libur sekolah mau habis dan saya mau masuk jenjang sekolah sma. Saya mengikuti tahfiz cabang 1 juz, pada saat saya belum serius menghafal sudah pasti saya kalah, namun bagusnya saya tidak dibabak pertama.

  • 5. Peserta Olimpiade INFORMATIKA SMA tingkat kabupaten

    Saat itu, saya berada dikelas 2 SMA yang lagi diadakan lomba olimpiade sma tingkat kabupaten cabang Informatika. Cabang Informatika berisi soal penalaran matematika, disini saya mencoba mengikutinya. Namun, belum berhasil mendapatkan juara. ketika tahun selanjutnya saya mencoba mengikutinya kembali namun tetap saja tidak membuahkan hasil yang baik. MUngkin, saya kurang dalam bidang tersebut.
